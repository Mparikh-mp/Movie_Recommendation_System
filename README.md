# Movie Recommendation System Using Machine Learning and Python

## Project Overview
This project is a hybrid movie recommendation system developed using Python and Machine Learning techniques.

## Features
- Content-Based Filtering
- Collaborative Filtering
- Hybrid Recommendation System
- Tkinter GUI
- Visualization Graphs
- Recommendation History

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Tkinter

## Dataset
- MovieLens Dataset
- TMDB 5000 Dataset
